# DoAn_WinForm – Pet Shop Management (WinForms, EF6, DevExpress v25)
Manage a pet shop with WinForms: products, spa/hotel services, a unified shopping cart, and invoice creation/printing. Three-layer architecture, Entity Framework 6, DevExpress v25.

# ✨ Features
- Products: CRUD, category filtering, inventory management.

- Services: Spa/Grooming — CRUD, seeded sample data.

- Pet Hotel: room map with statuses (Vacant / Occupied / Overdue).

- Unified Cart: aggregate products + services + pet-boarding from multiple forms.

- Invoices: create from cart, clear cart after checkout, reprint/view invoices.

- Administration: staff management, permissions/roles.

# 🧱 Technologies
- .NET WinForms (Framework 4.x)

- Entity Framework 6

- SQL Server (LocalDB/Express/Server)

- DevExpress WinForms v25.x

- Reporting: RDLC (if configured)
